Active development.

This minecraft client is being worked on actively. This means we add features you want into the client.

Active Discord.

The official discord server is an place where the developers are actiove in. So do you have an question or suggestion join the Discord and ask.

Opensource.

This client is completely opensource on the official github repo. This means that you can see all the code.

Configuration

Maintain all your settings in a single config.nbt file. Never need to redo the setting every single time opening the client. This also means that you can use someone elses config.

Does the client work with optifine?

Optifine currently doesnt work with MatHax Client and causes many issues with it. But you can use Lithium. This is an fabric optifine alternative.

How to open the Click Gui?

The Click Gui is binded to RIGHT_SHIFT and can be changed in the Controls settings.
