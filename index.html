Active development

This Minecraft Utility Client is being worked on actively. This means we add features you want into the client actively.

Active Discord

The official Discord server is an place where the developers are active in. So do you have an question or suggestion join the Discord and ask.

Open Source

This client is completely opensource on the official GitHub repository. This means that you can see all the code.

Settings

Maintain all your module settings in a single Modules.nbt file. Never need to redo the setting every single time opening the client. This also means that you can use someone elses settings.

Does the client work with Optifine?

Optifine currently doesnt work with MatHax Client and causes many issues with it. But you can use Sodium, Lithium or Phospor. Sodium is the best Fabric Optifine alternative.

How to open the Click Gui?

The Click Gui is binded to RIGHT_SHIFT and can be changed in the Controls settings.

How can i use it on 2b2t?

You can join any server with ViaFabric and ModMenu. In server list there will be a button with VIA on it. Open it and set Client-Side Mode to enabled & rewrite AUTO to version you want to join.

How to edit HUD?

There are gray TABS up in the middle of the screen. To edit it simply click on HUD tab.

How to disable Discord RPC?

There are gray TABS up in the middle of the screen. To disable it simply click on Discord Presence and unmark Enabled checkbox.
